Tech learnings:
    LLM:
        - An LLM is an AI model trained on a huge amount of text data (like books, websites, code, etc.) that can understand and generate human-like language.
        - Examples include:
            ChatGPT (by OpenAI)
            Gemini etc
        - What can they do?
            Summarize text
            Write code or emails
            Analyze documents etc
    Node.js:
        - Node.js is a JavaScript runtime environment that lets you run JavaScript code outside of a web browser, typically on a server.
    Why is that important?
        - Traditionally, JavaScript only ran in browsers to make websites interactive.
        - Node.js lets you write backend/server-side applications in JavaScript.

    So instead of using languages like Python, Java, or PHP for backend code, you can use JavaScript for both frontend and backend.

Vad är Middleware för något?
    - Middleware är ett begrepp inom programmering och systemutveckling som beskriver en mjukvarukomponent eller ett lager som ligger mellan två andra delar i en applikation eller ett system och fungerar som en slags "mellanhand".
        - Vad gör middleware?
            - Hantera kommunikation och dataflöde mellan olika delar, till exempel mellan klient och server.
            - Bearbeta, filtrera eller ändra inkommande förfrågningar och utgående svar.
            - Utföra gemensamma funktioner som autentisering, loggning, felhantering, säkerhet, datakonvertering med mera.
            - Förenkla och strukturera koden genom att dela upp ansvaret i separata lager.

Vad är 'context' för något?
    - "context" refers to the surrounding information, state, or environment in which something is happening.
       It helps a program or function "understand" how it should behave depending on where or how it's used.
    - Here in react, it is used as a share state across multiple components.
